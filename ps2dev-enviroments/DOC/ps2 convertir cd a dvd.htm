[Manual] Pasar Juego De : "CD A DVD"
* JUEGOS DE PSX (PS1) NO SE PUEDEN CAMBIAR A DVD

Lean cada uno de los pasos y seguro les resulta

Necesitamos:
- la iso del juego (obviamente en el pc).
- Drivers Aspi.
- Ps2 CDVDCheck 1.31 o DVDScan v3.6
- CD Dvd Rom Generator 1.5
- CDVDGEN ISOBuilder
- programa de grabación nero , alcohol,etc

Todo estos programas
AQUI

Ó AQUI

antes que nada recomiendo instalar los driver aspi y reiniciar el pc

1º¿Cómo obtener la imagen iso y extraerla a una carpeta?

si tienen el juego en cd exploren el cd y copian todas las carpetas al pc (copiar y pegar creando una carpeta para este, obvio no)
para los juegos bajados de internet pueden montar una unidad virtual y copiar todas las carpetas al PC

2º Parchear para DVD.

recomiendo quitar a la carpeta del juego el atributo solo lectura (click derecho propiedades )

para parchar a dvd vamos a usar Ps2CDVDCheck 1.31 este programa, nos parchea los archivos necesarios como los irx, img o elf (sea SLUS, SLES o SLPM) para no tener problemas al pasarlos de cd a dvd y asegurarnos que corra bien en DVD.

hagan lo siguiente
- Ejecutar el Ps2CDVDCheck 1.31.
- Seleccionamos la opción “parchear para funcionar en dvd”.
- Buscar la carpeta donde está el juego y seleccionar los archivos abriendo una por una las carpetas y parchándolas con este programa háganlo todo por separado para que estén seguros que les resulte
Nota: Si en el juego que estamos grabando no hay directorios, entonces lo que hacen es simplemente seleccionar todos los archivos marcarlos y ahí le ponen “parchear”

otra opción seria usar dvdscan v3.6
-Ejecutar DVDScan v3.6.
-Vamos a opciones y marcamos la casilla Auto-patch media checks con Yes, luego Done para que guarde los cambios.
-Elegimos DVD en la pantalla principal, abrimos los archivos que nos interesan y parcheamos para funcionar en DVD.



3º Crear la iso para nuestro juego en dvd.

- Ejecutamos el programa Sony cd dvd generator 1.5 donde dice CD_DVD-ROM Generator 1.50.
- Seleccionamos Create new Project y despues DVD-Rom master disc.
- Por defecto se abrirá en la pestaña Directory.

This image has been resized. Click this bar to view the full image. The original image is sized 706x465.


- Seleccionamos y arrastramos todos los archivos y carpetas que están en nuestra carpeta de juego sobre Directory del CD_DVD-ROM Generator 1.50.

This image has been resized. Click this bar to view the full image. The original image is sized 864x467.


- Ahora seleccionamos la pestaña Volume.

Dentro de Master Disc Information buscamos Disc Name y ponemos el identificador del juego (elf), por ejemplo, si fuera SLUS_999.9 escribiríamos SLUS 9999, sin guion bajo ni puntos, en la primera casilla las letras, en la segunda los números. Es muy importante no equivocarse en esto
- En Producer Name poner PLAYSTATION.
- En License Area poner lo que corresponda
- Ahora dentro de Volume Identifier buscamos Volume y Volume Set donde colocaremos el nombre que queramos, pero tiene que ser el mismo, algo que identifique el juego.



- Ahora clickeamos sobre la pestaña Layout. Aca se darán cuenta que hay una lista de archivos en rojo y en negro, en el primero de la lista de color negro, pulsar el botón derecho y elegir Location, ahora cambiamos el valor del campo por 600001. (Esto siempre si o si, no importa cual sea el juego, siempre el mismo numero)

This image has been resized. Click this bar to view the full image. The original image is sized 641x462.


- Ahora apretamos el botón Record, y nos va a pedir guardar un archivo iml (donde creara dos archivos un ims y un iml), guárdalo en otra carpeta . Después comúnmente nos da dos errores, no se preocupen, esos son normales, si llegara a dar uno mas eso depende de si tenemos instalados el pack Aspi, si todo salio bien se abre el programa cd/dvd-rom record unit controller versión 1.5.

This image has been resized. Click this bar to view the full image. The original image is sized 642x461.


-creamos una nueva carpeta

Ahora dentro de este programa abrimos el menú File y elegimos Export Image, y eligen la carpeta recién creada (todo esto de las carpetas solo para mantener el orden de los archivos y que no se confudan luego) hasta aquí todo bien



-Cierren todos los otros programas.

5ºJuntando los archivos para nuestra ISO final.

-Utilizar el programa CDVDGEN ISOBuilder para crear la imagen, tomando el archivo xxx.000, xxx.001, etc para crear la imagen. (no tomar el archivo xxx.lst).





- Paciencia, una vez listo estará en la ultima carpeta creada con un tamaño mayor de 1 Gb.

6º La grabación.
Grabar la iso con su programa favorito en el pack va incluido el prog. Dvd decrypter con su parche al español (aprovechen que me costó mas que la chu…… encontrarlo asi) y que es gratis
Recuerden grabar siempre a menos de 4x
saludos espero les sirva

PD: no todos los juegos se pueden cambiar de formato ó hay algunos que te permiten hacer todo el proceso pero finalmente quedan en pantalla negra ó se quedan pegados.
pero si lo quieren intentar adelante lo unico pueden perder es un dvd
